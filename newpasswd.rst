SYNOPSIS
========

::

    newpasswd [-h] [-l LEN] [-n N] [-r]

OPTIONS
=======

--help, -h              Show this help message and exit.
--length=LEN, -l LEN    Length of your password.
--count=N, -n N         Ammount of passwords to generate.
--random, -r            Use /dev/random as source of randomity.
